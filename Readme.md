# Telegram Bot

## Overview

This Telegram bot is designed to interact with users by processing text messages, images, and documents. It leverages the Gemini API for generating responses and DuckDuckGo API for performing web searches. The bot can also extract text from images and PDF documents.

## Features

- **Text Interaction**: Responds to user messages using the Gemini API.
- **Image Processing**: Extracts text from images using OCR (EasyOCR) and provides descriptions.
- **PDF Handling**: Extracts text from PDF files and summarizes their content.
- **Web Search**: Performs web searches using the DuckDuckGo API and returns summaries and links.
- **User Management**: Collects user information such as phone numbers.

## Usage

1. **Start the Bot**: Open Telegram and search for your bot using its username. Click "Start" to initiate interaction.
2. **Send Messages**: Type any message to receive a response from the bot.
3. **Send Images**: Upload an image to extract text or receive a description.
4. **Send Documents**: Upload a PDF document to extract text and get a summary.
5. **Web Search**: Use the command `/websearch <query>` to perform a web search.
